# Student Grade Manager (C)

## Overview

A console-based student management application developed in C that manages student records, calculates GPA, assigns grades, generates statistics, and stores data persistently using CSV files.

## Features

* Add student records
* View all students
* Search students by name
* Delete student records
* Automatic GPA calculation
* Automatic grade assignment
* Sort students by GPA
* Generate class statistics
* CSV-based persistent storage

## Technologies Used

* C Programming
* Structures
* Arrays
* CSV File Handling
* Sorting Algorithms

## Compilation

```bash
gcc grade_manager.c -o grade_manager
```

## Execution

```bash
./grade_manager
```
